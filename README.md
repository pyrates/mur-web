# MUR website

## Install

    make develop

## Init DB

    mur init-db

## Run

    mur serve --reload

Then go to http://localhost:1919


## Deploying

https://roll.readthedocs.io/en/latest/how-to/advanced/#how-to-deploy-roll-into-production

Configure any needed var form `config.py` by settings env var `MUR_XXX`.
